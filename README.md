### 超级重要：一定要看

1.本仓库对应同济大学数据结构课程设计选题中**尾号为 4** 的题目：邻接矩阵、产品装配

2.项目仅为前端项目，不需要后端

3.运行项目：

```bash
# 下载对应依赖
npm i
# 启动项目
npm run serve
# 打包项目
npm run build
```

4.数据结构课程设计展示形式**建议为 web 端**（默认要求QT），界面会更好看一些，**感觉得分会高一些**

5.项目名字是随便起的，**代码仅供参考**
